Panorama Stitching

Date: 03-01-2020 - 04-01-2020
Github

This project uses computer vision techniques(Harris Corner detector, SIFT, RANSAC) in order to stitch together images to create a panorama.

The process
  1. Takes 2 images and finds the keypoints using the Harris Corner Detector(By looking at the gradients of the image to find distinct corners)
  2. Uses the SIFT descriptor to describe each of these keypoints and compares each images descriptors to find matches using sum of squared differences and ratio of ssd to make sure there were no ambiguous matches
  3. Next the RANSAC algorithm is used which randomly selects 4 matches to calculate the homography. It then checks the amount of inlier matches with this homography. This algorithm runs over 1000 times and keeps the homography that gives the greatest number of inliers.
  4. The new image is stitched together by creating a shell of a new image that will fit image 1 and the transformed image 2. Image 1 is copied into the shell and the second image is transformed using the inverse homography into the shell.
  5. Repeat! 1-4 are repeated until all the images have been added to the panorama by using the last stitched image as one of the images in the next iteration.

Autonomous Defense Robot

Date: 11-15-2019 - 12-01-2019

Designed an autonomous robot to drive and avoid objects and trigger defense mechanism.

The details

This was the final project for my Embedded Systems class. The robot was built using two arduinos, a BeagleBone and necessary hardware(Level shifts, H-Bridges, Resistors, battery, sensors ). Sensors were connected through the arduinos which would signal the Beaglebone when the robot got too close to an object. The Beaglebone would then communicate to the arduinos to reverse the wheels and change direction. Everything was coded using bare metal C on the arduinos and python on the Beaglebone. The micro controllers communicated using SPI and the device tree on the beagle bone was manually configured for communication.
